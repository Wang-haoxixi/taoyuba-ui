# 国脉前端 [查看更新日志](./CHANGELOG.md)

## 技术栈
vue + vuex + vue-router + ES6/7  + sass + echarts + [element-ui](http://element.eleme.io/#/zh-CN) + vue-cookie + axios + [avue](https://avue.top/) + [lodash](https://www.lodashjs.com/docs/4.17.5.html) + [viser-vue](https://viserjs.github.io/)

## 上手开发
```bash
# 切换到分支 在分支开发
npm checkout dev
# 下载
npm install
# 启动项目
npm start
```
